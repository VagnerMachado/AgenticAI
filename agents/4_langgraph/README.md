# LangGraph Projects

This directory contains labs and applications built using the LangGraph framework.

## Labs

The labs are a series of Jupyter notebooks that introduce the core concepts of LangGraph.

- **1_lab1.ipynb**: Introduction to LangGraph and its basic building blocks.
- **2_lab2.ipynb**: Focuses on creating stateful and cyclical graphs.
- **3_lab3.ipynb**: Explores more advanced features like conditional edges and tool usage.
- **4_lab4.ipynb**: Covers building multi-agent systems with LangGraph.

## Applications

### Sidekick

This is a sample application demonstrating a conversational AI sidekick built with LangGraph.

- **app.py**: The main entry point for the Sidekick application.
- **sidekick.py**: Defines the agent and its graph.
- **sidekick_tools.py**: Contains the tools that the Sidekick agent can use.
- **memory.db**: A database file used for storing conversation history.
