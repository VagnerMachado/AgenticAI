# AutoGen Projects

This directory contains labs and applications built using the Microsoft AutoGen framework.

## Labs

The labs are a series of Jupyter notebooks that introduce different aspects of AutoGen.

- **1_lab1_autogen_agentchat.ipynb**: Introduction to creating multi-agent conversations with `AgentChat`.
- **2_lab2_autogen_agentchat.ipynb**: More advanced `AgentChat` scenarios, likely involving more complex agent interactions.
- **3_lab3_autogen_core.ipynb**: Focuses on the core components and lower-level abstractions of AutoGen.
- **4_lab4_autogen_distributed.ipynb**: Explores how to set up and run AutoGen agents in a distributed environment.

## Applications

### Agent Framework

This directory includes a set of Python files that constitute a basic framework for creating a multi-agent system.

- **agent.py**: Defines the base agent class.
- **creator.py**: A script or module for creating and configuring agents.
- **world.py**: The environment where the agents interact.
- **messages.py**: Defines the message structures used for communication between agents.
- **tickets.db**: A database file, likely used for task management or message storage.
